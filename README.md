# DKK MapLibre Workshop

Dieses Repository enthält die Materialien für den Workshop "DKK MapLibre Workshop". Es umfasst die Präsentationsfolien, das Handout, Beispielcode und weitere Ressourcen, die während des Workshops verwendet werden.

Den Demo-Code für die Code-Along Sessions finden Sie im file `index.html`
Alle Materialien finden Sie im Ordner `docs`.

- Die Präsentationsfolien unter `docs/slides.pdf`
- Das digitale Handout unter `docs/handout.pdf`
